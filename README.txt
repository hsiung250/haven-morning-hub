高天通訊處早會課程｜GitHub Pages 上傳版

【第一次上傳】
1. 將這個資料夾內的所有內容上傳到 GitHub repository 根目錄：
   - index.html
   - data.js
   - assets 資料夾
2. GitHub → Settings → Pages
3. Source 選 Deploy from a branch
4. Branch 選 main，Folder 選 /(root)，按 Save

【修改課程、姓名或標題】
1. 在 GitHub 點開 data.js
2. 點右上角鉛筆圖示 Edit
3. 修改引號內文字
4. 按 Commit changes
5. 等候約 1–3 分鐘後重新整理網站

【更換 Logo】
1. 新 Logo 檔名必須是 haven-logo.png
2. 放進 assets 資料夾，覆蓋舊檔
3. Commit changes

【重要】
- 請勿刪除 index.html。
- data.js 的逗號、引號與大括號不要任意刪除。
- 若只改中文文字，最安全。
